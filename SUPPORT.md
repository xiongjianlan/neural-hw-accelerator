# Support This Project 🤝

Thank you for your interest in the Neural Network Hardware Accelerator project! 

## 🌟 About This Project

This is a **complete open-source implementation** of a CNN hardware accelerator in Verilog, designed for:
- **Edge AI devices** - Low power consumption (34.6 mW)
- **Real-time video processing** - 11.1M windows/sec throughput
- **Educational use** - Complete example for hardware AI
- **Research** - Baseline for hardware neural network research
